# Changelog

## v0.1.0

Initial release.

### Features

- Live Claude Code agent visualization as pixel art characters
- 6 Metro City character sprites with 4-directional walk, typing, and reading animations
- BFS pathfinding on a 12x10 tile office grid
- Click-to-inspect info panel (model, branch, task, elapsed time)
- Sub-agent visualization with parent-child connection lines and spawn/departure animations
- "+ Agent" button to spawn new Claude Code terminals with optional task prompts
- Sound notifications on agent completion
- Dark pixel art UI theme
